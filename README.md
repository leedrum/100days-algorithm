# 100 days Algorithm

Just practice. 1 days = 1 algorithm, but for some algorithms, the cost (time) will be longer.

Details of each algorithm will be presented in the `README.md` of each folder.

## Running test cases

Each algorithm has a file `test.py`.

- Example:
```
python ./python/hanoi_tower
```
